# NewCool Mineria - SERNAGEOMIN Chile

Portal educativo sobre mineria y SERNAGEOMIN en Chile.

## Demo

[https://mineria.newcool.io](https://mineria.newcool.io)

## Caracteristicas

- Buscador de oficinas SERNAGEOMIN
- Guia de concesiones mineras
- Calculadora de patentes mineras
- Tramites de exploracion
- Normativa del sector

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
